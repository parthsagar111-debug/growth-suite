How this works

Every number on every page is computed by deterministic code — never guessed by a model. AI's job is layered on top: multiple named agents per tool interpret, compare, and narrate those numbers, and every tool shows its full dashboard before showing a recommendation.

The one deliberate exception is grading: SHIP/KILL/EXTEND is decided by comparing actuals against committed thresholds — by code, not by a model.

First Response is the one exception to the pattern above — no data upload, no shared memory, just a branching diagnostic chat that tells you what to check before you touch the other four tools.
